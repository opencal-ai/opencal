# OpenCal

API-first calorie & nutrition tracker. Beautiful iOS app, open API.

Tell your AI agent what you ate. It logs everything — calories, macros, meals. You never open the app.

## Install the skill

```bash
npx skills add opencal-ai/opencal@skill
```

Then give your agent your API key (Profile → API Keys → Generate in the app).

## What the skill does

Your agent can:
- Search foods and log meals ("I had 200g chicken breast for lunch")
- Check daily progress and remaining macros
- Update calorie/macro goals
- Delete or edit log entries
- Get meal suggestions based on your remaining macros
